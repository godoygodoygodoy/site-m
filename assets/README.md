# Assets Directory

Esta pasta contém os recursos estáticos da aplicação:

- `logo.png` - Logo da aplicação (opcional)
- Imagens dos livros são carregadas de URLs externas (Amazon)
- Fotos de perfil dos usuários (futuro)

## Notas

As imagens dos livros são carregadas de CDNs externos. Para usar imagens locais:

1. Baixe as imagens dos livros
2. Salve nesta pasta
3. Atualize os caminhos em `js/data.js`
